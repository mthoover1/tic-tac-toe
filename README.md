Tic Tac Toe
===========

Ruby Tic Tac Toe app with a focus on test-driven-development.

Prerequisites
-----
- Unix system

Setup
-----
- Clone

Play
-----
'ruby tic_tac_toe.rb'

Instructions
-----
- Enter board size ("3" would mean a 3x3 board)
- Human is "X", computer is "O"
- Refer to numbered diagram above empty board for tile numbers before moves

Running Specs
-----
'rspec'
